CrystallographyQLPlugins

A set of QuickLook plugins for Apple OSX, that provide rapid previews of files that are commonly found in structural biology

Welcome to GitHub Pages.

Structural biology is plagued with an excess of data, such as PDB files and X-ray diffraction images. We have written two independent QuickLook plugins for the OSX operating system, one for viewing Protein Data Bank (PDB) files and the other for X-ray diffraction images.

The PDB plugin can use pymol to quickly render thumbnail images (See screenshot 1), while the heart of the plugin lies with the integration of Bosco Ho’s HTML 5 PDB viewer, Jolecule (See screenshot 2). Jolecule provides the user with a lightweight ’3d’ interactive viewer without the need for opening larger applications like PyMOL, Coot, Chimera or whatever chooses your fancy. Furthermore, we attempt to extract important metadata from the PDB file and visibly present for rapid identification of the structure. The X-ray diffraction image plugin is designed to open .img and .osc file formats, convert the file to a jpeg and cache it for later display. This is done using a custom version of the CCP4 diff2jpeg library that is built into the plugin (See screenshot 3). The plugins are currently available together with an installer or separately, but require manual installation. The plugins have been tested on OSX 10.6, 10.7 and 10.8. Unfortunately, jolecule will not function on 10.6 as webkit was not introduced until OSX 10.7
